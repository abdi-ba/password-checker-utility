# 🔐 Password Strength Utility (v1.0)

A lightweight, terminal-based security tool designed to analyze password complexity and provide real-time feedback on credential strength. 

## 🌟 Overview
This utility was developed as a specialized cybersecurity project. It evaluates passwords based on length, character variety (uppercase, lowercase, digits, and symbols), and common patterns. To protect the intellectual property of the analysis logic, the tool is distributed as a pre-compiled binary.

## 🚀 Installation & Usage (macOS/Linux)

To run the tool without needing to install Python or external libraries, follow these steps:

1. **Download:** Go to the [Releases](https://github.com/abdi-ba/password-checker-utility/releases) section of this repository and download `abdi-checker.zip` (or the `.bin` file).
2. **Extract:** If you downloaded a ZIP, double-click it to extract the `abdi-checker` file.
3. **Set Permissions:** Open your Terminal and navigate to your Downloads folder:
   ```bash
   cd ~/Downloads
   chmod +x abdi-checker
